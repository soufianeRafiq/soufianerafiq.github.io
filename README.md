# Portfolio — Soufiane Rafiq

Page personnelle responsive (projet Responsive Web Design, EPSI Toulouse — SN2).

## Contenu
- `index.html` — structure de la page (HTML sémantique)
- `style.css` — mise en forme responsive (mobile-first, CSS Grid + Flexbox, media queries, animations)
- `script.js` — interactions (menu burger, thème clair/sombre, lien actif, barre de progression, révélations)
- `image.jpg` — ma photo

## Technologies
HTML5, CSS3 (Grid & Flexbox, media queries), JavaScript (vanilla).

## En local
Ouvrir `index.html` dans un navigateur (tous les fichiers doivent rester dans le même dossier).

## En ligne
Hébergé gratuitement via GitHub Pages.
